## Build

1. Get a copy of XiTK code
  ```sh
  git clone --recursive https://github.com/xipki/xitk.git
  ```
  The option `--recursive` is required to checkout the submodules.

2. Compile and install the artifacts

  In folder `xitk`
  ```sh
  mvn clean install
  ```
